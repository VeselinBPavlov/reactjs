Title: Slider

Description: This is a reactjs app to slide episodes and show characters information.

Instructions:
	- Instal depencencies: npm install on server and on client side
	- Start client side: npm start	
	- Location: localhost: 3000
	- Start server side: node index.js
	- Location: localhost: 9999

Technologies (client):
	- Framework: ReactJS

Technologies (server):
	- Framework: Express.js
	- Database: none
	- ODM: none
